# ✈️ Airline Data Analysis

## 📌 Project Overview
This project analyzes airline operational data to identify ways to improve profitability and aircraft occupancy rates. The analysis focuses on ticket bookings, revenue trends, fare conditions, and aircraft performance using SQL and Python.

The goal is to discover insights that can help airlines optimize pricing strategies and increase seat occupancy.

---

## 🎯 Objectives
- Analyze airline ticket booking data
- Evaluate revenue trends over time
- Compare fares across different aircraft and classes
- Calculate aircraft occupancy rates
- Identify opportunities to increase airline profitability

## 📊 Analysis Performed
- Aircraft seat capacity analysis
- Ticket booking trend analysis
- Revenue analysis across aircraft types
- Fare comparison across classes (Business, Economy, Comfort)
- Occupancy rate calculation
- Revenue impact analysis with increased occupancy

---

## 📈 Key Insights
- Aircraft with lower ticket prices showed higher booking volumes.
- Occupancy rate directly impacts airline profitability.
- Pricing strategy plays a major role in ticket demand.
- Increasing occupancy by 10% could significantly improve overall revenue.
